# Interactive Excel Sales Dashboard with VBA Automation

## Overview
This project is an interactive Excel dashboard designed to analyze sales performance using Pivot Tables, Slicers, and VBA automation.

## Key Features
- Dynamic Pivot Tables for sales analysis
- Slicers for region-wise filtering
- VBA macros with checkboxes to switch reports automatically
- KPI tracking: Total Sales, Target Achievement %, Away from Target
- MIS-style dashboard for management reporting

## Tools & Technologies
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Slicers
- Excel Formulas
- VBA (Macros)

## Use Case
This dashboard can be used for sales analysis, MIS reporting, and performance monitoring in business environments.
